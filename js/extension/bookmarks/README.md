# 书签管理器

## 功能概述

这是一个基于 WXT + Vue + Element Plus 开发的现代化书签管理器扩展，提供了丰富的书签管理功能。

## 主要功能

### 1. 书签管理
- **添加书签**：支持手动添加和右键菜单快速添加
- **编辑书签**：修改标题、URL、文件夹、标签和描述
- **删除书签**：安全删除确认机制
- **拖拽移动**：在文件夹间拖拽移动书签

### 2. 文件夹管理
- **单层文件夹结构**：简化的文件夹组织方式
- **文件夹操作**：创建、重命名、删除文件夹
- **未分类文件夹**：自动创建，用于快速收藏

### 3. 标签系统
- **标签筛选**：侧边栏标签列表，点击筛选
- **标签管理**：添加、编辑书签时管理标签
- **标签统计**：显示每个标签的书签数量

### 4. Power Search 搜索
- **普通搜索**：直接输入关键词搜索书签
- **搜索引擎模式**：输入 `!` 或 `！` 触发搜索引擎选择
- **站内搜索**：支持特定网站的站内搜索

### 5. 右键菜单
- **快速收藏**：右键选择"快速收藏到未分类"
- **完整添加**：右键选择"用书签管理器收藏"，打开完整添加表单

### 6. 导入导出
- **HTML导出**：生成标准HTML书签文件，兼容其他浏览器
- **JSON备份**：完整备份所有数据，包括标签和描述
- **文件导入**：支持HTML和JSON格式导入
- **预览功能**：导入前预览文件内容

### 7. Favicon图标
- **自动获取**：自动获取网站图标
- **多重降级**：Chrome API → 备用API → 默认图标
- **缓存机制**：本地缓存图标，提高加载速度

## 使用方法

### 安装扩展
1. 构建项目：`npm run build`
2. 打开Chrome扩展管理页面
3. 开启开发者模式
4. 加载已解压的扩展程序，选择 `.output/chrome-mv3` 文件夹

### 基本操作

#### 添加书签
1. **主界面添加**：点击"添加书签"按钮
2. **右键快速添加**：在网页上右键选择"快速收藏到未分类"
3. **右键完整添加**：在网页上右键选择"用书签管理器收藏"

#### 管理书签
1. **编辑**：鼠标悬浮在书签上，点击编辑图标
2. **删除**：鼠标悬浮在书签上，点击删除图标
3. **移动**：拖拽书签到目标文件夹

#### 搜索功能
1. **普通搜索**：在搜索框输入关键词，按回车
2. **搜索引擎模式**：
   - 输入 `!` 或 `！` + 空格
   - 选择搜索引擎
   - 输入关键词，按回车

#### 标签筛选
1. 在左侧标签列表中点击标签
2. 主界面只显示包含该标签的书签
3. 点击"清除筛选"恢复显示所有书签

### 高级功能

#### 设置搜索引擎
1. 添加书签时，在标签中添加 `search` 或 `search_site`
2. `search` 标签：URL中包含 `${keyword}` 占位符
3. `search_site` 标签：网站根地址，用于站内搜索

示例：
- Google搜索：URL = `https://www.google.com/search?q=${keyword}`，标签 = `search`
- V2EX站内搜索：URL = `https://v2ex.com`，标签 = `search_site`

#### 导入导出数据
1. **导出HTML**：生成通用书签文件
2. **导出JSON**：完整备份所有数据
3. **导入文件**：拖拽或选择文件导入

## 技术架构

- **框架**：WXT + Vue 3 + TypeScript
- **UI组件**：Element Plus
- **存储**：Chrome Storage API
- **权限**：storage, contextMenus, notifications, tabs, activeTab

## 开发命令

```bash
# 开发模式
npm run dev

# 构建生产版本
npm run build

# Firefox版本
npm run dev:firefox
npm run build:firefox
```

## 注意事项

1. **数据安全**：建议定期导出备份数据
2. **权限要求**：需要存储、右键菜单、通知等权限
3. **浏览器兼容**：主要支持Chrome，Firefox需要额外配置
4. **性能优化**：大量书签时建议使用标签筛选功能

## 故障排除

### 常见问题
1. **图标不显示**：检查网络连接，图标会自动重试获取
2. **搜索不工作**：确保搜索引擎书签配置正确
3. **导入失败**：检查文件格式是否正确
4. **拖拽无效**：确保在文件夹区域内拖拽

### 重置数据
如需重置所有数据，可在浏览器扩展管理页面重新安装扩展。
